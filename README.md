# ATtiny85 USB Rubber Ducky

A compact USB HID device built using the ATtiny85 microcontroller that emulates keyboard input for automation, cybersecurity learning, and USB HID experimentation.

## Features

- USB HID keyboard emulation
- Low-cost ATtiny85-based design
- Custom keystroke payload execution
- Compact and portable hardware
- Suitable for cybersecurity labs and educational use

## Hardware

- ATtiny85 Microcontroller
- USB Connector
- 2 × 3.6V Zener Diodes
- 2 × 68Ω Resistors
- LEDs with current-limiting resistors
- Supporting passive components
